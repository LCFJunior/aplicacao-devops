# Aplicação DevOps

Esta é uma simples aplicação Node.js com Express.js.

## Como usar

1. Instale as dependências:
    bash
   ```
    npm install
   ```

3. Inicie o servidor:
    bash
   ```
    node src/app.js
   ```

## Endpoints

- ```/``` : Retorna a página index.html.
- ```/integrantes``` : Retorna uma lista de integrantes do grupo.
